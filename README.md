# 🌐 Neon Digital

**Fully responsive marketing website** developed with **HTML, CSS, Tailwind CSS, Vite** and JavaScript.  
Focused on performance, clean design, and smooth UX.

## 🚀 Tech Stack

- HTML5
- CSS3
- Tailwind CSS
- JavaScript (Vanilla)
- Vite
- Supabase (data)
- Fancybox (gallery)
